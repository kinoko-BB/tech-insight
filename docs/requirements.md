# コーディング試験問題：AI搭載型ナレッジベース「TechInsight」の構築

## 1. 試験の概要

技術記事データを基盤としたナレッジマネジメントシステム「TechInsight」を構築してください。本試験では、基本的なデータ管理機能（CRUD）の実装に加え、AIを用いた「セマンティック検索」の統合能力、および実用的なシステム設計能力を評価します。

## 2. 技術要件

- Backend: Python / FastAPI
- Frontend: Next.js / TypeScript / React
- Database: PostgreSQL
- AI/ML: 任意のモデルを用いたベクトル化（Embedding）の実装
- Infrastructure: Docker Compose で全サービスがワンコマンドで起動すること

## 3. 実装タスク

### タスク1：データマイグレーションと環境構築

提供された articles.csv（1,000行）を読み込み、データベースに保存するスクリプトまたはプロセスを作成してください。


※提供されたデータは1000件ですが実際には1万件ほどに増えると仮定してクエリ効率などを考えた上で設計、実装してください


Docker Compose を使用し、データベース、API、フロントエンドが連携した状態で起動するように構築してください。


docker compose upでマイグレーションが行われ、Webアプリケーションが起動するまでを行われるようにしてください。

### タスク2：Backend APIの実装

CRUD API: 記事の取得（検索含む）、新規作成、更新、削除を行うエンドポイントを実装してください。
検索APIの詳細: ユーザーが入力した自然言語のクエリに対して、記事本文との「意味的な類似度」に基づいた検索結果を返す機能を実装してください。使用するモデルや類似度の計算手法、検索のロジックは各自の判断に任せます。

### タスク3：Frontendの実装

記事一覧・検索画面: 記事を一覧表示し、キーワードまたはセマンティック検索を実行できるUIを構築してください。
記事詳細表示: 記事の全内容を確認できるモーダル、あるいはそれに準ずる表示機能を実装してください。
管理機能: 記事の投稿、既存記事の編集および削除ができるインターフェース（モーダルやフォーム）を実装してください。
